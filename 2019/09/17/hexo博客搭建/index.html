    • 一、本地环境搭建

    1. node.js 下载安装
      官网：https://nodejs.org/en/
      点击安装包安装即可
      可以通过版本号来验证
      切换root用户：
sudu su
验证：
node -v
npm -v
    2. hexo 下载安装

    （1）更改使用淘宝镜像源cnpm

    npm install -g cnpm --registry=https://registry.npm.taobao.org

    （2）利用cnpm安装hexo

    cnpm install -g hexo-cli
验证
hexo -v

    二. hexo 生成博客

    （1）创建一个博客文件夹

    mkdir blog
cd blog

    （2）初始化博客

    sudo hexo init

    （3）启动博客

    hexo s

    此时本地hexo博客已经创建完成

    常用四个命令：

    1. 清理  hexo c
2. 生成静态文件 hexo g
3. 启动 hexo s
4. 发布 hexo d

    三.hexo博客和github关联实现外网访问

    1. github 登录并新建仓库
    仓库名称：账户名.github.io
    1. hexo 安装git插件
    cnpm install --save hexo-deployer-git
    1. 设置._config.yml 文件，配置de’ploy为git
      deploy：
 type: git
 repo: github仓库路径
 branch： master
    2. 部署倒github上
    hexo d
输入账号密码即可
    1. 浏览器访问：https://账户名.github.io 访问博客

    四.hexo更换皮肤主题

    1. hexo官方主题页面：https://hexo.io/themes/
    2. 按照主题中的操作步骤依次操作即可完成主题更换
